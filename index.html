Stenographic System for Fast Programming

I want to create a stenographic system for programming faster. Ideally this system would not only be useful in programming contexts but would also be useful in any context where Stenotype might be useful. I want to develop the hardware for this system using a pi pico and cherry-mx keyboard switches. The inputs would be mounted on an ergonomically 3d printed structure that will allow it to lay flat on desk while also providing full range of motion for wrist and finger joint articulation.
